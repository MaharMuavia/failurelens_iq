## Diagnosis
The model evaluation relied on accuracy (0.93) as the primary headline metric on a severely imbalanced 90/10 churn dataset. This allowed a model that largely ignores the minority churn class (producing minority F1 of only 0.14 versus baseline 0.65) to appear successful. The evaluation methodology did not prioritize operationally relevant metrics such as minority-class recall, F1, precision, or confusion-matrix shape, which the DP-100 guidance explicitly warns against.
- Violated assumption: That high overall accuracy is a reliable indicator of model quality when the positive class of interest (churn) constitutes only 10 % of samples.
- Knowledge gap: Team lacks awareness that accuracy is a misleading headline metric on imbalanced data; DP-100 style evaluation requires explicit focus on minority F1, PR curves, false-negative cost, and threshold selection rather than default accuracy.
- Evidence strength: strong

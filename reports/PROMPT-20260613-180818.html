Root Cause Analysis

Root Cause: In a 90/10 imbalanced churn prediction task the XGBoost model achieves 93% accuracy by correctly classifying the vast majority class while almost completely failing on churners, driving minority F1 down from a baseline of 0.65 to 0.14. The evaluation methodology continued to treat accuracy as the headline metric and did not surface or act on the minority-class collapse, which is the exact anti-pattern described in the DP-100 skill guide and the Failure Taxonomy's Evaluation Methodology Failure category.

Violated Assumption: That overall accuracy is a reliable and sufficient performance indicator for a binary classifier when the operationally important class (churn) constitutes only 10% of the data.

Knowledge Gap: The team did not internalize that accuracy is misleading on imbalanced datasets and that minority F1, recall, precision, confusion-matrix shape, and threshold-aware metrics must be the primary evaluation targets (as explicitly called out in the DP-100 and taxonomy grounding documents).

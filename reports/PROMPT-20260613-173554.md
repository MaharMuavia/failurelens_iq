## Diagnosis
The XGBoost classifier was evaluated primarily on accuracy (0.93) in a 90/10 imbalanced churn dataset. This allowed the model to achieve high accuracy by mostly predicting the majority non-churn class while almost completely failing on the minority churn class (minority F1 dropped from baseline 0.65 to 0.14). No drift, data quality, or leakage signals were present. This matches the exact failure mode described in the DP-100 skill guide and failure taxonomy: accuracy is a misleading headline metric for rare classes, and proper evaluation must center on minority F1, recall, precision, and confusion-matrix shape.
- Violated assumption: That accuracy is a sufficient and reliable performance metric for a 90/10 imbalanced classification problem.
- Knowledge gap: The team did not internalize that accuracy can be 90+% while minority-class performance is near zero; they lacked the habit of prioritizing minority F1 / PR-AUC / threshold-aware metrics and confusion-matrix review when the positive class (churn) is operationally critical.
- Evidence strength: strong

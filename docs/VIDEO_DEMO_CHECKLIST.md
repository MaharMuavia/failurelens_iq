# Video Demo Checklist

- Start backend from a clean demo environment.
- Start frontend and open the app.
- Confirm `/health`, `/readiness`, `/iq/status`, and `/demo/run` work.
- Click Judge Demo before recording or refresh once to warm the cache.
- Show classification, root cause, historian, remediation, certification readiness, and Microsoft IQ Proof panel.
- Show at least one citation count and the local fallback or live Azure badge.
- Mention that demo mode works without Azure credentials.
- Mention that production mode uses Azure AI Search for live Foundry IQ grounding.
- Keep the recording under 2 minutes.
- Do not show `.env`, API keys, terminal secrets, or cloud credentials.
